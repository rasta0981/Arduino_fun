# Arduino_fun
Provides a learning and testing environment to better understand microcontrolers.
All code utilized here is free and publicly avaiable.
------------------------------------------------------------------------------------
1. temp_sensor.ino > utilizes dht11 hum/temp sensor to report serial data information
2. blink-2.ino > esp32 board connected to three random LEDs 
3. OLED_TEMP_AHT21.ino > esp32 connected to 1 inch OLED screen that displays temp/hum from aht21 sensor. Includes code to render bitmap image on screen to display temperature gauge. (see doc)
4. aht_Sensor.ino > esp32 board connected to improved aht21 senor. (see doc)
5. LoRa_radio.ino > LoRa transmitter connected to aht20 sensor. (see doc)
6. LoRa_radio_receiver.ino > LoRa receiver connected to 1 inch OLED display. (see doc)
